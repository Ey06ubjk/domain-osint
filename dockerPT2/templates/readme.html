Semi-Automated OSINT

De tool die is ontwikkeld, is bedoeld om OSINT te automatiseren die inspiratie haalt uit MITRE ATT&CK -> https://attack.mitre.org/tactics/TA0043/. De 10 reconnaissance technieken van MITRE:

1. T1595 Active Scanning: Using techniques like port scanning and network mapping to identify active devices and services.
2. T1592 Gather Victim Host Information: Collecting details about a victim's systems, such as operating systems and installed software.
3. T1589 Gather Victim Identity Information: Obtaining personal identifying information of individuals associated with a target organization.
4. T1590 Gather Victim Network Information: Gathering data on a victim's network configurations, connections, and infrastructure.
5. T1591 Gather Victim Org Information: Acquiring information about the target organization, such as roles, relationships, and affiliations.
6. T1598 Phishing for Information: Using deceptive emails or communications to elicit sensitive information.
7. T1597 Search Closed Sources: Searching through non-public databases or forums for information.
8. T1596 Search Open Technical Databases: Looking up technical data like certificates and network infrastructure in public databases.
9. T1593 Search Open Websites/Domains: Researching information available on public websites or domain registries.
10. T1594 Search Victim-Owned Websites: Investigating websites owned by the target for exploitable information.

De helft van deze methoden is digitaal. Daarom zijn alleen de digitale technieken 1, 3, 4, 8 en 9 in aanmerking genomen voor het product. Deze methoden worden gecategoriseerd als passieve of actieve reconnanssaince, afhankelijk van het type informatie dat we verzamelen en hoe we betrokken zijn bij de digitale infrastructuur van de target. Deze methoden kunnen natuurlijk ook worden gebruikt voor red teaming.

Omdat OSINT altijd menselijke input vereist, kan het nooit volledig worden automatiseerd. Maar het gebruik van dit product kan bepaalde processen versnellen. Om het beste resultaat van deze tool te krijgen, moet u de volgende stappen volgen:

1. Het gebruik maken van de Scanner.
2. Het grondig doornemen van de OSINT Tools.
3. Handmatig zoeken naar overige informatie volgens MITRE.
